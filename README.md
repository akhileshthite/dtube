## Development
Open ganache personal blockchain. Don't forget to add private key in MetaMask.

```
ganache-cli
```

Install dependancies

```
npm install
```

Deploy the smart contracts to the blockchain.

```
truffle migrate
```

Start react app.

```
npm start
```
